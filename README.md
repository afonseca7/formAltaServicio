# formAltaServicio
Diseñar y desarrollar un formulario web que permita la carga de datos de un usuario interesado en solicitar el alta de un servicio.

El formulario deberá contemplar, como mínimo, las siguientes secciones:

Datos personales
Datos de contacto
Dirección
Además, deberán definirse e implementarse las validaciones necesarias para asegurar la integridad y consistencia de la información ingresada.

El diseño del formulario deberá plantearse de la forma más genérica posible, de manera tal que pueda adaptarse a distintos rubros o tipos de servicios, evitando referencias excesivamente específicas a un único caso de uso.

Trabajo grupal.
Utilizar IA para el desarrollo. Se sugiere el uso de Copilot.
Programar la solución utilizando HTML, CSS y JavaScript.
Todo el desarrollo deberá estar contenido en un único archivo con extensión html.
El archivo deberá poder abrirse en un navegador web y permitir probar el formulario sin depender de archivos externos adicionales.
El formulario deberá ser funcional, claro y presentar una interfaz ordenada y comprensible para el usuario.
Utilizar la rúbrica adjunta para auto-evaluación.
